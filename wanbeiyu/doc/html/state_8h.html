Parses a 9-byte input and updates the state accordingly.

Parameters
bufferPointer to a 9-byte input array.
statePointer to the state structure.

This order is derived from devkitPro, and originally comes from the representation in memory on the actual console.

According to devkitPro and Citra, the console internally uses a range of -156 to 156 for the C Stick and -154 to 154 for the Circle Pad. The up-right direction is considered positive, with the center at (0,0). These values apply to the uncalibrated case; the actual range may vary depending on calibration.

A value of 1 indicates the component is held; 0 indicates it is released.
