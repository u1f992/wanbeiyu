wanbeiyu_uint16_t wanbeiyu_touch_screen_state_t::x

Horizontal coordinate.

The origin is at the left edge; X increases to the right.

This value is not constrained to the screen width (max 319). It is clamped by wanbeiyu_touch_screen_set before use.
